About This App

This is a simple weather application that fetches real‑time weather data from wttr.in, a free weather service.

  • Frontend: HTML, CSS, JavaScript (vanilla)
  • Backend: Node.js + Express
  • Weather API: wttr.in (no API key required)
  • Deployment ready: Includes Dockerfile and deployment guides
